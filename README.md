# E-commerce System (College Project)

This is a simple command-line shopping project made with Python and SQLite.

## What it does

- Register and login users
- Show product list
- Search products
- Add and remove items from cart
- Checkout with optional discount code
- Save and show order history

## Files

- `main.py` -> menu and program flow
- `storage.py` -> database setup
- `users.py` -> register/login
- `products.py` -> product functions
- `cart.py` -> cart functions
- `orders.py` -> checkout and orders

## Run

```bash
python3 main.py
```
